Settings & data source
The relay holds your Dhan token as a server secret — it never touches this app or GitHub. Worker code & setup are in dhan-relay-worker.js / README.md.
Heads-up: browsers usually block direct calls to api.dhan.co (CORS). Use this only in a WebView/browser without CORS enforcement. Relay mode is the reliable path.
CPCR = ΔPut OI ÷ ΔCall OI on the single ATM strike locked at 9:30 (PCR uses the ±N band). Prev close (default) measures the day's net OI change vs yesterday's close — the standard "change in OI", available from the first refresh. 9:30 baseline measures change only since today's 9:30 snapshot.
